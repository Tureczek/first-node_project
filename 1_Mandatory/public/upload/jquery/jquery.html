What is jQuery?


jQuery is a JavaScript library which was based on the principles, that you should be
able to do more while writing less. The jQuery library is to included on the HTML page
in the header section, in a script tag.

https://ajax.googleapis.com/ajax/libs/jquery/3.5.1/jquery.min.js

Jquery can be used for HTML DOM and CSS manipulation, and it includes AJAX which has the
$. notation.
The basic syntax of jQuery looks like the following:


$(selector).methodOrFunction();




Before adding jQuery to the page, it is a good idea to make sure that the the HTML document
and its elements are fully loaded. Otherwise there could be occur complications with the jQuery code.
The way to do this is by make a "document ready" event.
If we take the above code for example, where we pressed a button, and then it showed the URL to put
inside the script tag. We made sure that the document was ready before the jQuery function could be executed.


$(document).ready(function () {

$("#libButton").click(function () {

$("#jqSlideBox").slideToggle(2000);

});

});

But it is not necessary to use the document ready on every jQuery method. The little snippet
you see above is made by different jQuery methods, both for styling and placement. To make a box as the
one above.

First off we have the HTML code

Second we use some css to style a box, so that we can use the template multiple times




Then we use jQuery, to style the box on its id, we can see that by the "#" reference

Lastly we are pushing the text inside the box, to follow proper code syntax. Notice here that the (document).ready has not been called.
This shows that it is not necessary to have it, but still it is a good idea.
