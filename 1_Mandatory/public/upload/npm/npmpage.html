What is NPM?

NPM is a package manager for Node.js packages, or modules if you like.

www.npmjs.com hosts thousands of free packages to download and use.

The NPM program is installed on your computer when you install Node.js

NPM in the terminal

Check for version

Writing "npm -v" in the terminal will show if you have npm installed on your computer and if so, then what version is installed




NPM and express

What is express

Express is the most popular Node web framework, and is the underlying library for a number of other popular Node web frameworks.

Express can be installed through the terminal with "npm i express". - i is short for install.

It is important to note that, before you can install anything to your project through NPM, you need first to create a json file in your project, called "package.json".




When installed, you can see the dependency being added to package.json



Three ways to install npm dependencies in the package.json file

Manually install

Terminal install

Npm init install
